About Project

Sentiment Analysis

The code scrapes news headlines from Finviz for chosen stocks, then assesses sentiment using VADER Sentiment Analyzer. After aggregating sentiment scores, it employs a simple linear regression model to predict stock returns based on sentiment. However, the model's returns are randomized in this demonstration, requiring real historical data for practical application.

Random Forest

The Random Forest Regressor from scikit-learn's ensemble module is applied in this code to predict future stock prices using historical data. It utilizes features like opening price, high price, low price, volume, and sentiment score to generate predictions for a specific investment time horizon.

KMeans

KMeans clustering from scikit-learn's cluster module is utilized to group customers into three clusters based on their income and savings. This clustering helps categorize customers into distinct risk profiles for stock investments, enabling personalized recommendations tailored to their risk preferences.
